# HTML Learning Repository

Welcome to my **HTML Learning Repository** 
This repository contains all the concepts, examples, and mini-projects I’ve learned while studying HTML.

---

## About

This repo is created to:
- Learn the basics of HTML
- Understand web page structure
- Practice real-world examples
- Build a strong foundation for web development

---

## Topics Covered

- HTML Basics (Tags, Elements, Attributes)
- Page Structure (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Headings & Paragraphs
- Links & Images
- Lists (Ordered & Unordered)
- Tables
- Forms & Input Fields
- Semantic Tags (`<header>`, `<footer>`, `<section>`, etc.)
- `<div>` vs `<span>`
- Media (Audio, Video)
- Basic HTML Projects

---

##  Example Code

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>

    <h1>Hello World!</h1>
    <p>This is my first HTML page.</p>

</body>
</html>
